 Python Quiz Game
An interactive terminal-based quiz game built using Python that tests user knowledge through randomized questions, flexible answer input, and real-time score tracking.
📌 What this project does

This is a command-line quiz game where:
Questions appear in random order
You can answer using option number OR actual answer text
You get instant feedback after each question
Final score is displayed at the end
It’s basically a small project to practice logic building, user input handling, and clean program flow.

⚙️ Features

Randomized questions every time you play
Multiple-choice format
Accepts both:
Option number (e.g., 2)
Answer text (e.g., Canberra)
Immediate feedback (Correct / Wrong)
Final score calculation
Replay option
